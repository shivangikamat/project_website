# project_website

## Overview
This project is a web application built using React and Tailwind CSS. It includes various components and features to showcase projects interactively.

## Project Structure
```
index.html
package.json
postcss.config.js
README.md
tailwind.config.js
src/
	App.jsx
	main.jsx
	projects.js
	styles.css
	components/
		ProjectIsland.jsx
		ProjectRoom.jsx
```

## Features
- Interactive project display
- Modular React components
- Styled with Tailwind CSS

## Installation
1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd project_website
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage
To start the development server:
```bash
npm run dev
```

## Build
To create a production build:
```bash
npm run build
```
